#  proxy-list-random 1.0.1

you can get random list proxy and updated hourly

## Installation

Use the npm  to install foobar.

```javascript
npm i  proxy-list-random
```

## Usage

```javascript
const proxy = require(' proxy-list-random')

proxy().then(res => console.log(res))

```
